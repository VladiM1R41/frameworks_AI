
## Описание выбранных наборов данных

### 1. Набор данных для задачи классификации: Alzheimer's Disease Dataset

**Источник:** [Kaggle - Alzheimer's Disease Dataset](https://www.kaggle.com/datasets/rabieelkharoua/alzheimers-disease-dataset)

**Практическая задача:** Прогнозирование наличия болезни Альцгеймера у пациентов на основе медицинских, демографических и поведенческих факторов.


---

### 2. Набор данных для задачи регрессии: Student Performance Factors

**Источник:** [Kaggle - Student Performance Factors](https://www.kaggle.com/datasets/lainguyn123/student-performance-factors)

**Практическая задача:** Прогнозирование итогового балла студента на экзамене  на основе социально-экономических, образовательных и поведенческих факторов.

## Выбранные метрики качества

### Для задачи классификации (Alzheimer's Dataset):
1.  **Accuracy:** Доля верно предсказанных диагнозов от общего числа пациентов. Базовая и интуитивно понятная метрика.
2.  **Precision:** Доля верно предсказанных больных среди всех случаев, которые модель отнесла к классу "болен". Важна для оценки, насколько мы можем доверять положительному диагнозу от модели (минимизация ложных срабатываний).
3.  **Recall:** Доля верно предсказанных больных среди всех реально больных. Важна для оценки способности модели найти всех больных (минимизация пропусков заболевания).
4.  **F1-Score:** Гармоническое среднее между Precision и Recall. Позволяет найти баланс между двумя этими метриками, что особенно важно при дисбалансе классов.
5.  **ROC-AUC:** Оценивает способность модели разделять два класса (здоров/болен) независимо от выбранного порога классификации. Универсальная метрика качества бинарного классификатора.

### Для задачи регрессии (Student Performance Dataset):
1.  **MAE:** Средняя величина абсолютных ошибок. Измеряется в тех же единицах, что и целевая переменная (баллы), легко интерпретируется.
2.  **MSE:** Средняя величина квадратов ошибок. Сильнее штрафует за большие ошибки, что часто полезно.
3.  **RMSE:** Корень квадратный из MSE. Также измеряется в единицах целевой переменной, что облегчает интерпретацию по сравнению с MSE.
4.  **R²:** Показывает, какую долю дисперсии целевой переменной объясняет модель. Значение 1 - идеальная модель, 0 - модель, предсказывающая только среднее значение. Лучшая метрика для оценки объясняющей силы модели.
